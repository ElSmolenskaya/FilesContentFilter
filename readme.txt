    Технические характеристики проекта:

1. Openjdk version "20.0.1" 2023-04-18
2. Использована библиотека org.apache.commons.lang3, версия 3.14.0
   Источник: https://commons.apache.org/lang/download_lang.cgi


    Утилита фильтрации содержимого файлов:

    Утилита по очереди считывает строки из текстовых файлов, сортирует их и записывает в результирующие файлы
в соответствии с типом данных: вещественные числа в файл floats.txt, целые - в integers.txt,
строки - в strings.txt. Строки из файлов читаются по очереди в соответствии с их перечислением в
командной строке. В качестве разделителя используется перевод строки. Пустые строки будут проигнорированы.
Пробелы в начале и в конце строк будут удалены.
    Утилита работает только с числами в десятичной системе счисления, также возможна экспоненциальная
форма записи числа. Число в экспоненциальной форме всегда считается вещественным. Разделителем целой
и дробной частей всегда является ".". Число, содержащее ",", будет считаться строкой. Число не в десятичной
системе счисления будет считаться строкой.
    При запуске утилиты можно указать путь к результирующим файлам. Если путь не указан, результирующие
файлы будут записаны в текущую директорию.
    Также можно указать префикс, который будет добавлен к именам результирующих файлов.
    По умолчанию результирующие файлы перезаписываются, но можно выбрать режим добавления в
существующие файлы. Если во входящих файлах отсутствовал какой-либо тип данных, то соответствующий
результирующий файл создан или до/пере-писан не будет.
    По результату работы утилиты в консоль выводится статистика по каждому типу данных. По умолчанию
выводится только количество записанных строк в каждый из файлов. При выборе опции полной статистики для
строк дополнительно выведется длина самой короткой и длина самой длинной строки. Для числовых форматов
дополнительно будут выведены минимальное, максимальное числа, сумма всех чисел и среднее арифметическое
всех чисел, записанных в файл.
    При подсчете статистики для вещественных чисел, все числа в экспоненциальной форме будут переведены
в стандартную числовую запись. Все выведенные в статистику числа будут записаны в стандартной форме,
без округления. Среднее арифметическое число будет округлено до 20 знаков после точки как для целых,
так и для вещественных чисел.


    Инструкция по запуску утилиты фильтрации содержимого файлов:

1. Для установки утилиты выберите нужную директорию на вашем ПК или создайте новую
2. Скопируйте файл FilesContentFilter.jar в выбранную директорию
3. Запустите Командную строку на вашем ПК (консоль или терминал)
4. В командной строке наберите команду:
java -jar FilesContentFilter.jar param1 param2 param2_value param3 param3_value [file_name1] [file_name2]
 где:
    [file_name1],[file_name2] - имена входных файлов, содержимое которых необходимо отсортировать
    param1, param2, param3 - параметры запуска утилиты
    param2_value, param3_value - значение параметров запуска утилиты (для которых оно предусмотрено)

Имена входных файлов могут быть указаны с абсолютными или относительными путями. Если путь не указан,
то считается, что файл находится в текущей директории.

Параметры запуска утилиты:
    -o - путь для результатов; далее через пробел следует указать путь, где будут созданы (или до/пере-записаны)
        файлы с результатами работы утилиты
    -p - префикс для файлов с результатами; далее через пробел следует указать значение префикса
    -s - в консоль будет выведена короткая статистика по результатам работы утилиты (данная опция работает по умолчанию)
    -f - в консоль будет выведена полная статистика по результатам работы утилиты
    -a - устанавливает режим добавления в существующие файлы (по умолчанию файлы перезаписываются)

При установке взаимоисключающих или повторяющихся параметров будет учитываться значение последнего параметра.
